# Imports

Use this directory for raw source material and curated import summaries.

## Layout

```text
imports/
|-- raw/
`-- summaries/
```

## Rules

- Store raw exports only when needed for traceability.
- Prefer curated summaries.
- Do not store secrets or sensitive personal data.
- Delete raw imports when they are no longer useful.
- Link summaries to records created from the import.
