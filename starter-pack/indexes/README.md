# Indexes

Use this directory for generated or manually maintained indexes.

Suggested files:

- `organizations.md`
- `repositories.md`
- `topics.md`
- `tools.md`
- `people.md`
- `records.md`
- `records.json`
- `followups.md`
- `memory-summary.md`
- `stale-candidates.md`
- `agent-task-queue.md`
- `agent-task-graph.md`

If you do not have automation yet, keep simple Markdown lists here and update them when important records are added.

Generated indexes are read surfaces. Rebuild them from Markdown and do not make them the only place where facts live.
