# Tools

Use this directory for tools, frameworks, libraries, runtimes, model versions, and platform notes.

Recommended structure:

```text
tools/tool--<slug>/README.md
tools/tool--<slug>/records/yyyy-mm-dd--version--<subject>.md
```

Capture:

- version adopted
- reason for adoption
- compatibility notes
- verification command
- related repositories

Runnable starter-pack tools:

- [`tool--test-plan/`](tool--test-plan/) - local QA test-plan generation from
  tickets, Braingent memory, and optional Gather Step evidence.
