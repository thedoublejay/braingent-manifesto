# People

Use this directory only for professional collaboration context that helps future engineering work.

Rules:

- Keep notes minimal and respectful.
- Avoid sensitive personal data.
- Capture decisions and preferences, not gossip.
- Prefer project or interaction records when possible.

Recommended structure:

```text
people/person--<slug>/README.md
people/person--<slug>/records/yyyy-mm-dd--interaction--<subject>.md
```
